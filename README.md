# Regen Studio Demos

Interactive demonstration portal for [Regen Studio](https://www.regenstudio.world). Showcases prototypes exploring digital product passports, regulatory compliance, and civic innovation.

## Demos

| Demo | Description |
|------|-------------|
| **CPR DPP Tracker** | Construction Products Regulation — Digital Product Passport readiness tracker |
| **Battery Questionnaire** | EU Battery Regulation compliance self-assessment |
| **DPP System** | Digital Product Passport platform prototype |
| **EDI Wallet** | European Digital Identity Wallet concept |
| **AI Tax → UBI** | Universal Basic Income simulator |

## Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript (no framework, no build step)
- **Backend:** Supabase Edge Functions (Deno/TypeScript)
- **Hosting:** GitHub Pages

## Local Development

```bash
python3 -m http.server
# or
npx http-server
```

## Deployment

Push to `main` — GitHub Pages auto-deploys.
